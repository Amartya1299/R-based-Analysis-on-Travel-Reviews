# TripAdvisor Travel Preference Analysis

This project analyzes TripAdvisor user review data across multiple travel categories
to identify behavioral patterns and preferences.

## Objectives
- Exploratory data analysis of travel ratings
- Correlation analysis across destination categories
- Hypothesis testing using t-tests
- K-means clustering of travelers
- Logistic regression to identify factors linked to being a Beach Lover

## Dataset
- Source: TripAdvisor reviews (East Asia destinations)
- Records: 980 users
- Features: 10 rating categories (0â€“4 scale)

## Methods Used
- Descriptive statistics
- Correlation heatmaps
- Two-sample t-tests
- K-means clustering
- Logistic regression (binomial)

## How to Run
1. Clone the repository
2. Open `Analysis.R` in R or RStudio
3. Run the script (CSV path is relative)

tidyverse, corrplot, cluster, factoextra
